# 🧠 Insights Overview

### 📊 Customer Spend Patterns  
- Middle-aged customers generate highest revenue  
- Young adults shop most frequently  

### 🛍 Category Performance  
- Clothing is the strongest revenue category  
- Accessories & footwear follow  

### 🧾 Subscription Insights  
- Very low subscription adoption → retention opportunity  

### 💸 Discount Behavior  
- Discounted products lead to strong conversions  
- Many discount users still spend *above average*  

### 🎯 Segmentation  
- Returning + loyal customers = best long-term value
